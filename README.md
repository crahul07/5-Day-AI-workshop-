# 🚀 5-Day AI Workshop Repository  

Welcome to the **5-Day AI Workshop** repository!  
This project contains all the **learning materials, code, and practical implementations** covered during a 5-day hands-on AI workshop.

---

## 📌 Overview  

This workshop is designed to provide a **strong foundation in Artificial Intelligence and Generative AI**, combining both **theoretical concepts and hands-on coding**.

- Duration: **5 Days**
- Focus: **AI + Generative AI + Practical Implementation**
- Format: Learning + Coding + Mini Projects

---

## 🧠 What You Will Learn  

- Fundamentals of Artificial Intelligence  
- Large Language Models (LLMs)  
- Prompt Engineering Techniques  
- Embeddings & Vector Databases  
- AI Agents and Automation  
- Domain-Specific AI Models  
- MLOps for AI Deployment
